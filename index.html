Multi-Agent Stock Market Simulation

This is a multi-agent stock market simulation which employs a genetic algorithm in order to generate increasingly effective trading strategies for each agent. The simulation consists of 27 autonomous agents who each create 5 random trading strategies at the beginning of the simulation - each strategy is generated by randomly assigning a weight to a random number of common stock market indicators of the 10 made available to the agents. The simulation takes place over 3650 days, or ten years.

Read more about the simulation and see the source code on github!

  • Language: Java
  • Date: Spring 2015
  • Source Code: Github

MIPS Binary Encoding

This program, written in MIPS, takes a text file of syntactactically correct MIPS commands as input. The file is parsed, each instruction is encoded in 32bit binary format, and a text file containing the encoded instructions is produced. For example, if a line in the input text file read [add $t0, $s1, $s2], the corresponding line in the output would read [000000 10001 10010 01000 00000 100000].

Read more about MIPS instruction encoding and see the source code on github!

  • Language: MIPS Assembly
  • Date: Fall 2014
  • Source Code: Github

n-Queens

The eight queens puzzle is the problem of placing 8 queens on an 8x8 chess board such that no two queens threaten each other. This project provides a general solution to the problem (placing n queens on an nxn board) using brute force recursion, printing out all possible solution for any size board.

  • Language: Java
  • Date: Spring 2015
  • Source Code: Github

Float Interpreter

The float datatype is a single-precision 32-bit IEEE 754 value. This simple program takes any float as user input, and returns the Hex value and IEEE 754 binary format of the given float.

  • Date: Fall 2014
